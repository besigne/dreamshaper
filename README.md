# dreamshaper

> CRUD Simples de Cursos e Estudantes

## Tech Stack
- **Backend:** Laravel
- **Database:** MySQL
- **Frontend:** Vue.js

## Instalação

> Primeiro suba os container da aplicação, vão ser 3 no total, banco de dados, backend & frontend, nessa ordem:
```
docker compose up --build -d
```
> Rode as migrations junto com os seeders para ter informação para manipular:
```
sh backend/run.sh
```

## Todos os comandos devem ser rodados da raiz do projeto
